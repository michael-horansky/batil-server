# v0.1.0 : 18 Nov 2025
- First online release to [siqo.sk/batil](http://www.siqo.sk/batil/)
- The basic Batil package features
    - a centralised one-v-one matchmaking with a live service operating Batil matches
    - a custom board creator
    - a brand-new rating system
    - a text guide
    - a set of interactive tutorials
- This is a beta version intended for public playtesting and features a multitude of rule versions which the players specify before creating a challenge
- A [feedback form](https://forms.gle/yvaYjF7mAqE5y4vK9) is linked in the front page of the guide
